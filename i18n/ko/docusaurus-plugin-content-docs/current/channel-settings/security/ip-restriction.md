# IP 주소 제한

<p class="_text-wrapper_1eo3m_1">IP 주소 제한 기능은 엔터프라이즈 플랜 이용 시 추가 구독하여 사용 가능한 보안 패키지 기능입니다. '보안 패키지' 기능을 별도로 선불 충전 후 사용이 가능합니다. (<a href="https://root.channel.io/home" class="_hyperlink_1bfbt_1" target="__blank">→ 가격 문의하기</a>)</p>

<h3 class="_heading_19a6y_1 _h3-laptop_19a6y_37">IP 차단 기능이란?</h3>

<p class="_text-wrapper_1eo3m_1 _paragraph_1udmy_1">허용된 IP 주소로만 채널에 접근할 수 있습니다. 허용 목록에 IP 주소를 추가해 부정 로그인을 예방하고 보안을 강화할 수 있어요.</p>

<li class="_list-item_1fabq_1"><p class="_text-wrapper_1eo3m_1">최대 50개의 IP를 등록할 수 있으며, 단일 IPv4 및 CIDR 형식을 지원합니다.</p></li>

<li class="_list-item_1fabq_1"><p class="_text-wrapper_1eo3m_1">채널별 허용 IP 주소와 역할별 허용 IP 주소를 모두 설정할 수 있어요.</p></li>
